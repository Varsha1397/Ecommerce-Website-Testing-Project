# Ecommerce Website Testing Project

## 🌐 Application Under Test

Website Name: Eric Javits E-commerce Website  
URL: https://ericjavits.com  

The application is an online fashion e-commerce platform offering designer hats, bags, and accessories.

## 📖 Project Overview

This repository contains a **Manual Testing Project** performed on an E-commerce website. The objective of this project was to validate website functionality, usability, and responsiveness across **desktop and mobile platforms**.

The testing activity simulates real-world QA testing practices followed in software companies.


## 🎯 Testing Objectives

* Verify core e-commerce functionalities
* Validate UI/UX behavior
* Ensure responsive design across devices
* Identify functional and usability defects
* Improve overall user experience

---

## 🧪 Testing Scope

### Website Testing (Desktop)

* Homepage validation
* Product listing verification
* Search functionality
* Add to Cart
* Wishlist validation
* User Login & Logout
* Account management
* Navigation & menu validation
* Product filtering & sorting
* Cross-browser compatibility testing

### Mobile Testing

* Responsive UI validation
* Mobile Chrome browser testing
* Icon visibility verification
* Product listing behavior
* Search and navigation testing
* Layout alignment and usability checks
